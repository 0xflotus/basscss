Getting Started

Download or install from one of these sources

Download CSS Download Source

Clone from Github:

git clone https://github.com/jxnblk/basscss.git

NPM:

npm install basscss

Bower:

bower install basscss

CDN:

<link href="http://d2v52k3cl9vedd.cloudfront.net/basscss/5.2.1/basscss.min.css" rel="stylesheet">
