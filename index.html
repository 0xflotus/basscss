BASSCSS

Next-level CSS

Minimal, utilitarian, responsive, mathematical, rhythmic

BASSCSS is a lightweight collection of simple type and layout utilities that can be used as the basis for any web project. Use it out of the box or as the base for a larger SASS project.

Built around the base unit of 1rem, everything in BASSCSS should line up neatly to a visual rhythm.
