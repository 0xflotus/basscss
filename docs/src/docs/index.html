Getting Started

Download or install from one of these sources

Download CSS Download Source

Clone from Github:

git clone https://github.com/jxnblk/basscss.git

NPM:

npm install basscss

Bower:

bower install basscss

CDN:

<link href="http://d2v52k3cl9vedd.cloudfront.net/basscss/<%= version %>/basscss.min.css" rel="stylesheet">

Basscss is organized according to four high-level categories:
base elements, immutable utilities, layout modules, and color styles.
