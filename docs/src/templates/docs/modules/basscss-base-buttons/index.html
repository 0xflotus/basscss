Basscss Base Buttons

Base button styles module for Basscss

basscss.com

Structural styles and resets for buttons are set in this base element styles module. All buttons share common padding and height, well suited for tap targets on touchscreen displays. Color styles in other modules and other custom extensions adjust their surface-level appearance.
