Floats

These utilities are available in mobile-first breakpoint-based variations.

Prefix Description
(no prefix) Not scoped to a breakpoint
sm- --breakpoint-sm (default: min-width 40em)
md- --breakpoint-md (default: min-width 52em)
lg- --breakpoint-lg (default: min-width 64em)
.left { float: left } .right { float: right } @media (--breakpoint-sm) { .sm-left { float: left } .sm-right { float: right } } @media (--breakpoint-md) { .md-left { float: left } .md-right { float: right } } @media (--breakpoint-lg) { .lg-left { float: left } .lg-right { float: right } }
