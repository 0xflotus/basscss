Layout

Use the following layout utilities to alter default HTML layout.

.inline { display: inline } .block { display: block } .inline-block { display: inline-block } .overflow-hidden { overflow: hidden } .clearfix:before, .clearfix:after { content: " "; display: table } .clearfix:after { clear: both } .fit { max-width: 100% } .half-width { width: 50% } .full-width { width: 100% }
